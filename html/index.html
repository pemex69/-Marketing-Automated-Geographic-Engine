Este es el landing page de //MAGE

El objetivo de este proyecto es ayudar a pequeñas empresas o pequeños emprendedores a ubicarse en un punto estratégico para iniciar y/o escalar un negocio y mejorar su presencia en el mercado tanto como aumentar su número de ventas

Para ello, se utilizarán datos recolectados de INEGI y de Google Maps para analizar la ubicación de los negocios y la ubicación de los clientes potenciales


El proyecto se divide en 3 partes:

  • El análisis/recolección de datos
  • La creación de un modelo de datos
  • La creación de una aplicación web

El análisis/recolección de datos se realizará en Python, utilizando las librerías de Pandas y Numpy para el análisis de datos y Matplotlib para la visualización en CMD

El modelo de datos se realizará en MySQL (en realidad es PostgreSQL lmao)

La aplicación web se realizará en NodeJS preferiblemente, utilizando Bootstrap para el diseño y ChartJS para la visualización de los datos (si se puede)


El proyecto se encuentra en desarrollo, por lo que no hay mucho que ver por aquí

¿Qué es Locationwise?

Locationwise es una aplicación web que te ayuda a encontrar el lugar perfecto para tu negocio, basándose en datos de la población y de los negocios de tu zona.

¿Cómo funciona?

Locationwise te muestra un mapa con los negocios de tu zona, y te permite filtrarlos por tipo de negocio, así como por la cantidad de clientes potenciales que hay en tu zona.

¿Cómo puedo usarlo?

Para usar Locationwise, primero debes registrarte en la página, y después iniciar sesión. Una vez que hayas iniciado sesión, podrás ver el mapa con los negocios de tu zona.
